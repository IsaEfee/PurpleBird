# Unity 2D - Flappy Bird Clone 🐦

A classic 2D Flappy Bird clone built from scratch using the **Unity Game Engine** and **C#**. This project was developed to practice core game development concepts, physics-based movement, and UI management.

## 🎮 Features
* **Physics-based Movement:** Smooth jumping mechanics using Rigidbody2D.
* **Modern Input System:** Utilizes Unity's New Input System for responsive keyboard (Space) and mouse clicks.
* **Dynamic Difficulty:** The game progressively gets harder; obstacle speed increases as the player's score gets higher.
* **Infinite Spawning:** Obstacles are dynamically instantiated and destroyed outside the camera view for memory optimization.
* **Game Management:** Includes a robust GameManager for score tracking, audio handling, and Game Over/Restart UI states.

## 🛠️ Technologies Used
* **Game Engine:** Unity 2D
* **Language:** C#
* **Version Control:** Git & GitHub

## 🚀 How to Play
1. Download or clone the repository.
2. Open the project in Unity.
3. Open the `SampleScene` (or your main scene name) in the `Scenes` folder.
4. Hit the **Play** button.
5. Press the **Spacebar** or **Left Mouse Button** to make the bird flap and avoid the pipes!
